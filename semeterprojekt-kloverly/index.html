Semesterprojekt – Kløverly (JavaFX)

Dette semesterprojekt er udviklet i Java og JavaFX som en del af Software Engineering-uddannelsen og tager udgangspunkt i casen Kløverly, et bæredygtigt bofællesskab med fokus på grønne initiativer og fællesskab.

Projektets formål var at designe og implementere et system, der understøtter både beboere og administratorer i håndteringen af daglige opgaver og grønne handlinger. Systemet bidrager til struktur, overblik og motivation gennem et integreret pointsystem.

Applikationen giver mulighed for administration af beboere, opgaver og point, hvor beboere kan registrere udførte opgaver, og administratorer kan godkende disse og tildele point. Pointene anvendes til at synliggøre både individuelle og fælles bidrag i bofællesskabet.

Systemet er opbygget med fokus på objektorienteret design, klar struktur og brugervenlighed. Applikationen følger MVC-principperne og er understøttet af UML-diagrammer, som har været anvendt til at planlægge og dokumentere systemets arkitektur og funktionalitet.

Funktioner

  • Oprettelse og administration af beboere
  • Opgavestyring og godkendelse af udførte opgaver
  • Pointsystem til registrering af grønne handlinger
  • Oversigt over individuelle og fælles point

Teknologier og værktøjer

  • Java
  • JavaFX
  • MVC-arkitektur
  • UML-diagrammer
  • SceneBuilder
