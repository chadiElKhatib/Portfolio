Hvem er jeg?

Jeg hedder Chadi El-Khatib og er softwareingeniørstuderende ved VIA University College. Jeg arbejder målrettet på at udvikle mine færdigheder inden for softwareudvikling med fokus på web, backend og systemdesign.

Min tilgang

Jeg motiveres af at forstå, hvordan systemer hænger sammen - fra brugergrænseflade til logik, databaser og netværk. Jeg arbejder struktureret og fokuseret og lægger vægt på at udvikle løsninger, der både er funktionelle, brugervenlige og teknisk solide.

Hvad jeg arbejder med

  • Frontend: HTML, CSS og JavaScript
  • Backend: Java (objektorienteret programmering, struktur og logik)
  • Databaser: SQL og PostgreSQL (strukturering og håndtering af data)
  • Netværk: Forståelse for TCP/IP, klient-server arkitektur og socket-programmering
  • Systemdesign: Arbejde med arkitektur, lagdeling og design af større systemer

Projekter og erfaring

Jeg har arbejdet med flere projekter, herunder webprojekter samt udvikling af et større system i Java (StockTradingGame), hvor jeg har arbejdet med lagdelt arkitektur, datahåndtering og opbygning af systemlogik.

Derudover har jeg udviklet GUI-applikationer i JavaFX, hvor jeg har arbejdet med brugergrænseflader, navigation mellem views og opbygning af logik.

Mit mål

Mit mål er at udvikle mig til en stærk softwareudvikler, der kan arbejde med både frontend og backend og bidrage til udviklingen af stabile og skalerbare systemer i praksis.

Jeg søger en trainee- eller praktikplads, hvor jeg kan omsætte min viden til praksis og udvikle mig yderligere som softwareudvikler.

Jeg er motiveret for at lære nyt og udvikle mig hurtigt i et professionelt udviklingsmiljø.
